# Grub Theme Expedition 33

Un thème immersif pour le chargeur d'amorçage GRUB, inspiré par l'univers et la direction artistique du jeu *Clair Obscur : Expedition 33*.

---

## ✨ Fonctionnalités

* Design épuré et moderne basé sur l'esthétique d'Expédition 33.
* Icônes personnalisées pour la plupart des distributions Linux (Ubuntu, Arch, Fedora, Debian, etc.) et Windows.
* Police de caractères adaptée à l'univers du jeu.
* Support des résolutions 1080p (ajustez selon votre thème, ex: 1080p, 4K, ultrawide).

## 🛠️ Installation

### Prérequis
Assurez-vous que GRUB2 est bien installé sur votre système (c'est le cas par défaut sur la très grande majorité des distributions Linux).

### Méthode manuelle

1. **Cloner le dépôt** (ou téléchargez-le en .zip et extrayez-le) :
   ```bash
   git clone [https://github.com/VOTRE_PSEUDO/grub-theme-expedition33.git](https://github.com/VOTRE_PSEUDO/grub-theme-expedition33.git)
   cd grub-theme-expedition33
